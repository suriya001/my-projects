MCQ Test Creator 

Description:
	1)It is a web application which allows to create MCQ Test.
	2)Any number of tests can be created.
	3)The test creator can add as many questions as needed.
	4)The test creator can also specify as many options as needed for each question and the correct option.
	5)Each test will have a unique test url.
	6)Test creator controls the test by starting and finishing the test.
	7)Candidate can attend the test and can view the answers after completion of test.
	
Developed using:
	Servlet
	JSP
	JS
	AJAX
	HTML
	CSS
